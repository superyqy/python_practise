工具名：基于webdriver的UI自动化测试框架
作者：殷秀文
创建时间：2018-06-23
主要功能：采用PageObject模式，封装页面基本操作，封装实体页面基本操作，形成测试用例，并在UnitTest框架下运行测试用例并做断言。默认使用Firefox作为测试浏览器 
此处实现的用例主要功能是：search book> enter detail page> add into cart>gather price and text value> check result

应用场景：UI自动化测试

开发脚本相关：Python + Selenium Webdriver + UnitTest


前提：
系统已安装Firefox


脚本使用方法：  python run_testcase.py

脚本输出：
unittest断言结果
